# RPC 

**What is a procedure call?** \
a local procedure call is like a function call within a process to execute some code to do certains task.

**What is RPC?** \
A Remote Procedure Call enables one machine to invoke execution of some code within another machine.

## gPRC:
gPRC uses **json** or **protocol buffer** to encode/decode send the to remote machines.
