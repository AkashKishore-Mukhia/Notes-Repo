# File I/O

- File
- FileReader
- FileWriter
- BufferedWriter
- BufferedReader
- PrintWriter

## File class constructor
```java
1. File f = new File(string name)
2. File f = new File(string subdir, string name)
3. File f = new File(File subdir-ref, string name)
```

## Create file:
```java
public class Main {
    public static void main(String[] args) throws IOException{
      File f = new File("meta.txt");
      System.out.println(f.exists());

      f.createNewFile();
      System.out.println(f.exists());
    }

}
```

## create directory:
```java
public class Main {
    public static void main(String[] args) throws IOException{
      File f = new File("meta");
      System.out.println(f.exists());

      f.mkdir();
      System.out.println(f.exists());

      File f1 = new File(f, "metadata.txt");
      f1.createNewFile();

      String[] s= f1.list();
      System.out.println(s);
    }

}
```

## File Writer:
We can use File Writer to write char data to the file.

### constructors:
```java
1. FileWriter fw = new FileWriter(String fname);
2. FileWriter fw = new FileWriter(File f);
3. FileWriter fw = new FileWriter(String fname, boolean append);
4. FileWriter fw = new FileWriter(File f, boolean append);
```

### Methods:
1. write(int ch);
2. write(char[] ch);
3. write(String s);
4. flush(); // total data will be written to the file.
5. close();

## File Reader:
### constructors:
```java
1. FileReader fr = new FileReader(String fname);
2. FileReader fr = new FileReader(File f);
```

### Methods:
1. int read() -> return unicode value(-1 if not char present to read)
2. int read(char[] ch) -> it attempts to read enough char from file and copies to char array, returns #ofCharCopied

## Buffered Writer:
Buffered writer can't communicate directly with the file, it can via Writer object.
Buffered Writer handles the newline \n char insertion by it's own using newLine().

### constructors:
```java
1. BufferedWriter bw = new BufferedWriter(Writer w);
2. BufferedWriter bw = new BufferedWriter(Writer w, int bufferSize);
```

## Buffered Reader:
We can read data from file line by line using readLine() method.

### Constructors:
```java
1. BufferedReader br = new BufferedReader(Writer w);
2. BufferedReader br = new BufferedReader(Writer w, int bufferSize);
```
