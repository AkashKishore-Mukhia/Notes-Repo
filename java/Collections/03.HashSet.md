# Set

It represent a collection of unique element, it does not allow duplicate values.
- The set interface does not allow duplicate element.
- It can contain atmost `one null` value except **TreeSet** which does not allow null values.

## HastSet
> Note: The order of elements in HashSet is not guaranteed, so the elements will be displayed in a random order if any are added.


```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
      Set<String> s = new HashSet<String>();
      
      // add element to HashSet
      s.add("s");
      s.add("e");
      s.add("t");
      
      // Print the HashSet
      System.out.println("printing set using println expression:" + s);
      
      s.remove("e");
      System.out.println("size of the HashSet is: " + s.size());
      
      // Iterating throught the set using for-each loop
      System.out.println("Printing each element using for-each loop: ");
      for (String each: s) {
        System.out.println(each);
      }
      
    }
}
```
