# File and Directory Permission

**Structure of folder and files**
```sh
# example
$ drwxrwxr-x 7 akash akash 4096 Oct 17 01:07 Zotero_linux-x86_64
```
| d | rwx | rwx | r-x |
|---|-----|-----|-----|
| File type | user permission | group permission | other permission | 


**The first character of permission strings are:**
- d = directory 
- 1 = symbolic link 
- b = block device 
- c = character device 
- s = socket

| **permission bits for file**  | **permission bits for directory**  |
|-------------------------------|------------------------------------|
| r = read | r = read |
| w = write | w = write alter the content inside the dir |
| x = execute | x = permission to access the dir |


**changing owner for file and dir**
```sh
$ sudo chown <user-name> <file/directory>
```

**changing permission for file and dir**
```sh
$ sudo chmod <user(u)-group(g)-other(o)> <+|-> <r|w|x> <file/folder>
```

**numerical represantation of permission bits:**  
r = 4  
w = 2  
x = 1
