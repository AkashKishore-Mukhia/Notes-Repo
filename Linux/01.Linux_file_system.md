# Linux File System

- In linux everythings is represented as files.
- Linux filesystem is hierarchical tree like struction, where root node is `/`

`/dev` -> house device associated file/dir \
`/home` -> house all user home dir's \
`/boot` -> house all file/dir for boot process \
`/etc` -> house system wide configuration files/dir \
`/bin` -> house essential binaries for all users \
`/root` -> Home directory for root user \
`/var` ->  Contains files that application writes information to as they run on system. \
`/sbin` -> Stores essential binaries
