# Collection
group of individual object's as a single entity.

## Array vs Collection
|Array| Collection|
|-----|-----------|
|Fixed in size|growable|
|Memory waste is more| Memory waste is less|
|performance is more | performance is less|
|readimade method is not present| readymade method available|
|homogenious | heterogenious|
|can hold both priemtive and object | can hold only object type|

### Collection
- Group of individual objects as a single entity.
- collection interface defines most common method which are application for any collection object
- Root interface of collection framework
- There is not concreate class which implements collection interface directly

#### Collection vs Collections
|Collection|Collections|
|----------|-----------|
|interface | class|
|group of objects|utility class for several utility methods for collection objects|
