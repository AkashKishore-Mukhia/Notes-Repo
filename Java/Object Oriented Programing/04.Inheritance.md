# Inheritance
- Inheritance is also know as Is-a-relation
- Main advantage of inheritance is code reusability
- Using `extends` keyword we can implement a relation
  ```java
  class Animal {
    static void sound(String sound) {
      System.out.println(sound);
    }
  }
  
  public class dog extends Animal {
    public static void main (String[] args) {
      sound("vow vow...");
    }
  }
  
  // output:
  vow vow...
  ```
- Parent reference can hold child class object but can't access any child class memebers.
  ```java
  class Animal {
    static void sound(String sound) {
      System.out.println(sound);
    }
  }

  public class dog extends Animal {
    static Boolean isPet() {
      return true;
    }
    public static void main (String[] args) {
      Animal a = new dog();
      a.isPet();
    }
  }
  ```
- Java does't support for multiple and hybrid inheritance.
- Java does't support cyclic inheritance.
