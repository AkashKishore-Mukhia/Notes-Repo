# Packages

- It is a encapsulation mechanism to group related classes and interfaces into a single unit as packages.
- Purpose fo pacakges are:
    - We can resolve naming conflicts
    - Modularity of the application improves
    - Maintability of application improves
    - provide security for components

Example:
```java
package com.mycompany.xrot;

public class Test {
    public static void main(String[] args) {
        System.out.println("pkg demo");
    }
}

// javac Test.java -> generated .class will be placed in CWD
// javac -d . Test.java -> generated .class will be in corresponding package structure
```

`Note:` If more than one package statement is encounter java compiler will throw an error,
package is the first non comment statement(if it's available)
